# KiCad Conventions

All KiCad schematics for LibreBoard Zero should use this title block:

```scheme
(title "LibreBoard Zero")
(date "YYYY-MM-DD")
(rev "v1.0")
(company "LibreBoard")
(comment 1 "Zeyad Bekhet (zbekhet)")
(comment 2 "https://github.com/zeyadhost/libreboard-zero")
```

Notes:

* The date should be updated to the current date when creating/exporting a schematic.
* The revision should always stay `v1.0` for now.
* Apply this convention to any schematic files created from now on.
